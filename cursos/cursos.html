Front-end Back-end Frameworks

Front-end

Curso HTML

Aprende en este curso completo HTML de forma completa y gratuita para poder a empezar a estructurar de una forma profesional tu sitio web con todo tipo de estructuras.

Curso CSS

Si ya sabes como estructurar tu web con HTML, ahora se aproxima lo mejor, el diseño de tu web. Con CSS puedes decorar y posicionar elementos de tu web como quieras.

Curso JS

Ya que has aprendido a hacer tu página web estática profesional con HTML y CSS viene siendo hora de que el usuario pueda interactuar con tu web modificando elementos HTML de forma dinámica.

Back-end

Curso PHP

Si ya has aprendido todo lo necesario para saber montar el front end de tu web de forma profesional ahora puedes aprender PHP para poder manejar información en bases de datos como quieras.

Curso MYSQL

Este será en lenguaje con el que manejarás los datos en la base de datos de tu sitio web utilizando como conector un lenguaje de programación de back end como PHP.

Curso MONGODB

Este es otro tipo de herramiento para manejar los datos de tu web solo que tiene otras características

Frameworks

Curso BOOTSTRAP 4

Si ya sabes controlar CSS no te viene nada mal empezar a dominar algún framework CSS y Bootstrap mejor forma para hacer tus diseños de forma sencilla.

Curso ANGULAR

Framework de JS muy potente capaz de hacer diseños y crear funcionalidades de sitios web de forma profesional.

Curso DJANGO

Si quieres especializarte en la parte back end de sitios web, este framework es perfecto para poder manejar esa información con Python.
